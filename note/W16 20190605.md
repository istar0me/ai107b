# 馬可夫鏈(markov chain)
* 具有狀態的有限過程，類似於「有限狀態機」

```
P(babb)
= P(b) x P(b->a) * P(a->b) * P(b->b)
= 0.8 * 0.5 * 0.3 * 0.5
= 0.06
```
